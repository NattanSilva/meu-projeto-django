# meu-projeto-django
Meu primeiro projeto django na fabrica!

```shell
    # Criando venv
    python -m venv venv

    # Crie um gitignore e escreva /venv/ dentro

    # Ativar venv(Windows)
    .\venv\Scripts\activate

    # Ativar venv(linux e mac)
    source venv/bin/activate

    # Instalar Django
    pip install django

    # Criar requirements.txt
    pip freeze > requirements.txt

```
